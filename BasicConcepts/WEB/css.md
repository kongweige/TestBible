**创建css**
* 外部链接css文件：
  * rel属性：rel="stylesheet"表示被链接文档是一个样式表文件，用于定义文档的样式
  * type属性：type属性指定了被链接资源的MIME类型。text/css，它告诉浏览器如何解析所链接的文件。在HTML5中，type="text/css"是默认值，可以省略不写
```html
<link rel="stylesheet" type="text/css" href="styles.css">
```
